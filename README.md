# Online Course Recommendation System

## Project Overview

This project is a Hybrid Recommendation System designed to recommend relevant online courses to users. The system combines Content-Based Filtering and Collaborative Filtering techniques to provide accurate and personalized course recommendations.

## Features

* Content-Based Recommendation using course attributes
* Collaborative Filtering using user-course interactions
* Hybrid Recommendation Model combining both approaches
* Evaluation using Precision@K, Recall@K, Hit Rate@K, and Coverage
* Interactive user interface built with Streamlit

## Dataset

The dataset contains:

* User Information
* Course Information
* Ratings
* Enrollment Details
* Course Duration
* Difficulty Level
* Certification Availability
* Feedback Scores
* User Engagement Metrics

## Recommendation Techniques

### Content-Based Filtering

Recommends courses based on similarities between course features such as duration, difficulty, price, certification, enrollment, and feedback scores.

### Collaborative Filtering

Recommends courses based on user behavior and rating patterns using a course-user interaction matrix.

### Hybrid Recommendation

Combines Content-Based and Collaborative Filtering scores to generate more accurate recommendations.
